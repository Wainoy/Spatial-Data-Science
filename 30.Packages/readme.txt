packages are used to organize code further.
related modules can be stored in a package
a package is a directory/ folder. new > package
a package has the file __init__.py
to create a package:
	go to the working directory within pycharm environment and click new > package. a new directory with the specified file name is created. a file __init__.py is automatically created.
Within this package, several modules can be created. several functions can be stored in each of these modules.
A function can be accessed by importing the entire module or the specific function
